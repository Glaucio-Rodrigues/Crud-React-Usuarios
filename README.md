# Crud-React-Usuarios
 # SQL Usuarios

Sistema full-stack de gerenciamento de usuários (CRUD) construído com React, Node.js e banco de dados SQL Server.

## Principais Recursos
- **Interface:** React, Vite e Tailwind CSS.
- **API:** Node.js e Express.
- **Banco de Dados:** Microsoft SQL Server (`mssql`).
- **Diferenciais:** Validação de caracteres nos campos, reordenação automática de IDs ao deletar e busca (atualização/deleção) por ID ou Nome.
